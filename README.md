# 📊Finance_Management_System
🚀 Overview

This project is a backend system for a finance dashboard where users can manage financial records based on their roles. It focuses on clean API design, proper data handling, and role-based access control using Java and Spring Boot.

💡 Features
User management with roles (Viewer, Analyst, Admin)
Role-based access control using Spring Security
CRUD operations for financial records (income/expense)
Filtering by date, category, and type
Dashboard APIs for:
Total income & expenses
Net balance
Category-wise breakdown
Recent activity & trends
🛠️ Tech Stack
Java
Spring Boot
Spring Security
JPA / Hibernate
MySQL / H2
Maven
⚙️ Run Locally
git clone https://github.com/your-username/finance-backend.git
cd finance-backend
mvn spring-boot:run

🧠 Note

This project was built to demonstrate backend design, access control, and data processing in a simple and maintainable way.
